# Github Pages 시작

개발자로서 테크 블로그 하나쯤 해보고 싶다고 전부터 생각하던 차, 네이버나 티스토리는 뭔가 Geek함이 부족해보여 Github Pages가 지원하는 Jekyll을 사용해 블로그를 만들기로 했다.

글을 쓰는 것도 굉장히 오랜만이라 감회가 새롭다. 비단 테크 뿐만 아니라 관심 있는 분야 두루두루 블로깅할 예정이다.
